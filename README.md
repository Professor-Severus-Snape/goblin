# Игра «поймай гоблина»

Браузерная мини-игра на чистом JavaScript. 

Гоблин случайным образом на одну секунду появляется на игровом поле 4×4 — задача игрока успеть кликнуть по нему и заработать очки.

![](./src/img/example.gif)

## Цели проекта
1. Практика работы с DOM и событиями
2. Работа с таймерами (setInterval)
3. Разделение логики приложения на классы
4. Настройка инфраструктуры проекта с нуля (линтинг, сборка, CI/CD)

## Механика игры
- Поле 4×4 (16 ячеек), генерируется динамически
- Гоблин создаётся программно и перемещается каждую секунду
- Повторное появление в той же ячейке исключено
- Ведётся подсчёт попаданий и промахов
- По окончании игры выводится модальное окно с результатом 

## Реализация
- Архитектура построена на классах: логика игры, управление отображением, работа с DOM и событиями разделены по зонам ответственности.
